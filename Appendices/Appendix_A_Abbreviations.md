## Common Surgical Prefixes and Suffixes

| Prefix/Suffix | Meaning | Example |
|---------------|---------|---------|
| **-ectomy** | Surgical removal | Appendectomy, cholecystectomy |
| **-otomy** | Surgical incision into | Laparotomy, thoracotomy |
| **-ostomy** | Surgical opening/stoma | Colostomy, tracheostomy |
| **-plasty** | Surgical repair/reshaping | Rhinoplasty, arthroplasty |
| **-orrhaphy** | Surgical repair/suturing | Herniorrhaphy, tenorrhaphy |
| **-scopy** | Visual examination | Laparoscopy, cystoscopy |
| **-pexy** | Surgical fixation | Mastopexy, nephropexy |
| **hyster-** | Uterus | Hysterectomy |
| **cholecyst-** | Gallbladder | Cholecystectomy |
| **chole-** | Bile | Choledochotomy |
| **nephro-** | Kidney | Nephrectomy |
| **gastro-** | Stomach | Gastrotomy |
| **entero-** | Intestine | Enterotomy |
| **colo-** | Colon | Colostomy |
| **cardio-** | Heart | Cardiomyopathy |
| **neuro-** | Nerve | Neurosurgery |
| **osteo-** | Bone | Osteotomy |
| **arthro-** | Joint | Arthroscopy |
| **angio-** | Blood vessel | Angioplasty |
| **cysto-** | Bladder/cyst | Cystoscopy |
| **thoraco-** | Chest | Thoracotomy |
| **laparo-** | Abdomen/flank | Laparotomy |
