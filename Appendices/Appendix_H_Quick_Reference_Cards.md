# Appendix H: Quick Reference Cards

One card per major procedure. Everything you need to prep a case, fast.

**How to use:** Find the procedure. Read it the night before. Walk through the checklist before you scrub. Use the closure table when the surgeon asks what's next.

---

## Card H-01 · Laparoscopic Appendectomy

| | |
|---|---|
| **Wound Class** | II (Clean-Contaminated) |
| **Position** | Supine → slight left tilt |
| **Avg Time** | 30–60 min |
| **Blade** | #11 × 3 (port stabs) |
| **Scope** | 30° laparoscope |

**Before You Scrub**
- 30° scope + camera + light cord confirmed
- 10 mm clip applier loaded; 2 extra cartridges
- Endoloops × 3 (2-0 chromic)
- Specimen bag for extraction
- Irrigation: warm saline, 1L

**Closure**

| Layer | Suture |
|-------|--------|
| Fascia (10–12 mm ports) | 0 Vicryl CTX |
| Skin | 4-0 Monocryl or skin glue |

**Watch For:** Clip applier misfire — have loaded backup ready. Retrocecal appendix — longer dissection, more Bovie use.

## Card H-03 · Open Inguinal Hernia (Lichtenstein)

| | |

**Watch For:** Ilioinguinal nerve crossing the field — protect it. Mesh must lie flat, tension-free.

## Card H-04 · Open Colectomy

| | |
|---|---|
| **Wound Class** | II (Clean-Contaminated) |
| **Position** | Supine |
| **Avg Time** | 2–4 hrs |
| **Blade** | #10 (midline incision) |

**Before You Scrub**
- GIA 75/80 mm stapler + 2 extra loads (blue)
- TA 55 or 60 mm stapler for rectal cases
- Bowel bag (isolate the specimen)
- 3-0 Vicryl for mesenteric vessels; 0 Vicryl for anastomosis
- Irrigation: 3L warm saline + bulb syringe

**Closure**

| Layer | Suture |
|-------|--------|

| Subcutaneous | 3-0 Vicryl |
| Skin | Staples (Class II); or packed if Class III/IV |

**Watch For:** Bowel content spillage → upgrade to Class III. Count all staple loads fired and discarded.

## Card H-14 · Cystoscopy with Ureteral Stent

| | |
|---|---|
| **Wound Class** | II (Clean-Contaminated) |
| **Position** | Lithotomy |
| **Avg Time** | 15–30 min |
| **Blade** | None |
| **Fluoroscopy** | Required |

**Before You Scrub**
- Cystoscope sheath + bridge + telescope (30° + 70°)
- Double-J stent: confirm size ordered; unopened on field
- Guidewires: hydrophilic × 1; stiff × 1 backup
- Stent pusher matching the stent system
- Irrigation: 1L sterile saline × 2 primed
- C-arm sterile drape

**Closure**
No wound closure. Foley catheter may be left per surgeon preference.

**Watch For:** Irrigation bottle running dry → field collapses instantly. Guidewire resistance → do not force; different wire type needed.

## Card H-15 · TURP

| | |
|---|---|
| **Wound Class** | II (Clean-Contaminated) |
| **Position** | Lithotomy |
| **Avg Time** | 45–90 min |
| **Blade** | None |
| **Irrigation** | Glycine 1.5% (monopolar) OR saline (bipolar) — confirm before spiking |

**Before You Scrub**
- Resectoscope: sheath (22–26 Fr) + working element + telescope — matching system
- Irrigation fluid type confirmed with surgeon; do NOT mix up glycine and saline
- Ellik evacuator on field
- 3-way Foley (22–24 Fr, 30 mL balloon) ready for end of case
- Track irrigation in vs. irrigation out throughout

**Closure**
No wound closure. 3-way Foley placed with continuous bladder irrigation post-op.

**Watch For:** TURP syndrome (glycine absorption) — deeply negative fluid balance = emergency. Do not resect past the verumontanum.

## Card H-17 · Tonsillectomy & Adenoidectomy (T&A)

| | |
|---|---|
| **Wound Class** | II (Clean-Contaminated) |
| **Position** | Supine, shoulder roll, neck extended |
| **Avg Time** | 20–45 min |
| **Blade** | None (ESU or coblation) |
| **Airway** | Shared (RAE tube) |

**Before You Scrub**
- Crowe-Davis mouth gag: correct size (peds vs. adult); suspension arm available
- Tonsil clamps (Allis or Hurd) × 2
- ESU pencil or coblation wand — confirm generator settings
- Tonsil sponges counted; specimen cups × 2 (left tonsil, right tonsil)
- 3-0 chromic on a tonsil needle if suture hemostasis needed

**Closure**
No formal closure. Mucosal beds heal by secondary intention.

**Watch For — CRITICAL:** Every tonsil sponge placed in the pharynx must come out before gag removal. A retained tonsil sponge = airway obstruction. Count out loud. Confirm before removing the gag.

## Card H-19 · Cataract (Phacoemulsification + IOL)

| | |
|---|---|
| **Wound Class** | I (Clean) |
| **Position** | Supine, head in donut |
| **Avg Time** | 15–30 min |
| **Blade** | Keratome (2.2–2.8 mm) + paracentesis knife |
| **Implants** | IOL (confirm power + type) |

**Before You Scrub**
- IOL loaded in injector per manufacturer instructions — do NOT touch with gloves
- OVD (viscoelastic) drawn and on Mayo stand
- Phaco machine: tubing primed; BSS bags spiked; footpedal set
- Eye speculum: correct size
- Dilating drops confirmed given 30–60 min pre-op; confirm full dilation
- Every solution labeled; do NOT use CHG near the eye

**Closure**
Self-sealing keratome incision — no sutures. Subconjunctival antibiotic + steroid injection. Rigid eye shield (not a pressure patch).

**Watch For:** IOL contamination → trash it; open new one. Posterior capsule rupture → case changes; different instruments needed. Label every solution on the back table.

## Card H-20 · Split-Thickness Skin Graft (STSG)

| | |
|---|---|
| **Wound Class** | I–II |
| **Position** | Per wound location |
| **Avg Time** | 30–90 min |
| **Blade** | Dermatome (powered or manual) |
| **Two sites** | Donor site + recipient site |

**Before You Scrub**
- Dermatome: blade guard depth set per surgeon specification; mineral oil on field
- Mesh carrier (if meshing): correct expansion ratio assembled
- Damp saline gauze on back table for graft storage
- Bolster or NPWT dressing confirmed
- Donor site dressing: Adaptic/Xeroform + gauze + ACE
- Change outer gloves between donor harvest and recipient site

**Closure**
Donor site: non-adherent dressing only — NOT sutured. Graft secured with staples + bolster or NPWT.

**Watch For:** Graft desiccation → keep it moist at all times. Hematoma under graft = graft failure. Achieve complete hemostasis in the recipient bed before graft goes down.

## Card H-23 · CDH Repair (Congenital Diaphragmatic Hernia)

| | |
|---|---|
| **Wound Class** | I (Clean) |
| **Position** | Supine, slight contralateral rotation |
| **Avg Time** | 60–120 min |
| **Blade** | #15 (subcostal incision) |
| **Patient** | Neonate (days old); left side most common |

**Before You Scrub**
- Neonatal instrument set (baby-scaled only)
- 2-0 or 3-0 Prolene/Ethibond interrupted for primary repair
- Gore-Tex patch on field if large defect expected
- Warming blanket active; warm irrigation; overhead warming lamp if available

| Subcutaneous | 3-0 Vicryl |
| Skin | 5-0 Monocryl or skin glue |

**Watch For:** Temperature every 15 min — a 2°C drop is a physiological emergency. Herniated organs are fragile; handle with extreme gentleness during reduction.

## Card H-24 · Free Flap (Microvascular)

| | |
|---|---|
| **Wound Class** | I (Clean) |
| **Position** | Varies by flap and defect |
| **Avg Time** | 4–10 hrs |
| **Blade** | #15 both sites |
| **Two teams** | Donor harvest + recipient site simultaneously |

**Before You Scrub**
- Microvascular instruments: Castroviejo needle drivers × 2; jewelers forceps × 2; micro scissors; vessel background material
- 9-0 or 10-0 nylon on BV-1 needle × multiple (keep moist)
- Venous coupler system (confirm sizes)
- Sterile Doppler probe
- Operating microscope drape
- Communication: both surgeons must confirm readiness before pedicle division

**Closure**
Donor site: layered primary closure or STSG. Recipient: non-compressive window dressing only — no pressure on the flap.

**Watch For:** Ischemia time begins at pedicle division — have everything ready before that moment. Compression on the flap after anastomosis = thrombosis = flap loss.

## Quick Reference: Closure Suture by Layer

| Layer | Standard Suture | Common Alternatives |
|-------|----------------|---------------------|
| Fascia (abdomen) | 0 PDS running loop | 0 Vicryl running |
| Fascia (orthopedic) | #1 Vicryl | 0 Ethibond |
| Peritoneum | 3-0 Vicryl running | Often left open |
| Muscle (deep) | 2-0 Vicryl interrupted | 0 Vicryl |
