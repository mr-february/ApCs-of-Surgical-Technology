    Appendix H: Quick Reference Cards¶

    One card per major procedure. Everything you need to prep a case, fast.

    How to use: Find the procedure. Read it the night before. Walk through the checklist before you scrub. Use the closure table when the surgeon asks what's next.

    Card H-01 · Laparoscopic Appendectomy¶
    Wound Class II (Clean-Contaminated)
    Position Supine → slight left tilt
    Avg Time 30–60 min
    Blade #11 × 3 (port stabs)
    Scope 30° laparoscope

    Before You Scrub - 30° scope + camera + light cord confirmed - 10 mm clip applier loaded; 2 extra cartridges - Endoloops × 3 (2-0 chromic) - Specimen bag for extraction - Irrigation: warm saline, 1L

    Closure

    Layer Suture
    Fascia (10–12 mm ports) 0 Vicryl CTX
    Skin 4-0 Monocryl or skin glue

    Watch For: Clip applier misfire — have loaded backup ready. Retrocecal appendix — longer dissection, more Bovie use.

    Card H-02 · Laparoscopic Cholecystectomy¶
    Wound Class II (Clean-Contaminated)
    Position Supine, reverse Trendelenburg + right tilt
    Avg Time 30–60 min
    Blade #11 × 4 (port stabs)
    Fluoroscopy For cholangiogram (if performed)

    Before You Scrub - 30° scope + camera + light cord confirmed - 10 mm clip applier loaded; 2 extra cartridges - Cholangiogram catheter + C-arm sterile drape if IOC planned - Maryland dissector + hook electrocautery on field - Specimen bag

    Closure

    Layer Suture
    Fascia (10–12 mm) 0 Vicryl CTX
    Skin 4-0 Monocryl

    Watch For: Critical view of safety must be confirmed before clipping. CBD clipping = bile duct injury. Biliary leak: suction ready.

    Card H-03 · Open Inguinal Hernia (Lichtenstein)¶
    Wound Class I (Clean)
    Position Supine
    Avg Time 45–75 min
    Blade #10

    Before You Scrub - Polypropylene mesh (Prolene; confirm size ~6×11 cm) - 0 Prolene for mesh fixation; 2-0 Prolene for cord sling - 1-0 PDS for internal oblique/transversalis repair - Penrose drain as cord retractor (½ inch) - 3-0 and 4-0 Monocryl for subcutaneous/skin

    Closure

    Layer Suture
    External oblique 3-0 Vicryl running
    Scarpa's fascia 3-0 Vicryl
    Skin 4-0 Monocryl subcuticular

    Watch For: Ilioinguinal nerve crossing the field — protect it. Mesh must lie flat, tension-free.

    Card H-04 · Open Colectomy¶
    Wound Class II (Clean-Contaminated)
    Position Supine
    Avg Time 2–4 hrs
    Blade #10 (midline incision)

    Before You Scrub - GIA 75/80 mm stapler + 2 extra loads (blue) - TA 55 or 60 mm stapler for rectal cases - Bowel bag (isolate the specimen) - 3-0 Vicryl for mesenteric vessels; 0 Vicryl for anastomosis - Irrigation: 3L warm saline + bulb syringe - Change gloves after bowel is opened

    Closure

    Layer Suture
    Fascia (midline) 0 PDS running loop
    Subcutaneous 3-0 Vicryl
    Skin Staples (Class II); or packed if Class III/IV

    Watch For: Bowel content spillage → upgrade to Class III. Count all staple loads fired and discarded.

    Card H-05 · Total Hip Arthroplasty (THA)¶
    Wound Class I (Clean)
    Position Lateral decubitus (posterior)
    Avg Time 60–90 min
    Blade #10
    Implants Shell + liner + stem + head

    Before You Scrub - Implant sizes: 2 above and below templated size - Oscillating saw + reciprocating saw; reamers in order - Trial components on back table in sequence - Double-glove; replacement outer gloves on field - Hemovac drain + 2-0 nylon for securing

    Closure

    Layer Suture
    External rotators #1 Ethibond through drill holes
    IT band / fascia lata #1 Vicryl
    Subcutaneous 2-0 Vicryl
    Skin Staples or 3-0 Monocryl

    Watch For: Cement has a working window — track time from mix. Change outer gloves at implant placement.

    Card H-06 · Total Knee Arthroplasty (TKA)¶
    Wound Class I (Clean)
    Position Supine, tourniquet
    Avg Time 75–105 min
    Blade #10
    Implants Femoral component + tibial tray + poly ± patella

    Before You Scrub - All trial sizes pulled and ordered - Oscillating saw + extra blades - Cement: mixing supplies ready; do NOT mix until call - Tourniquet pressure confirmed with surgeon (typ. 250–300 mmHg) - Double-glove

    Closure

    Layer Suture
    Arthrotomy (quad/medial parapatellar) #1 Vicryl figure-of-eight
    Subcutaneous 2-0 Vicryl
    Skin Staples or 3-0 Monocryl

    Watch For: Gap imbalance → have thin and thick poly inserts ready. Track tourniquet time; alert surgeon at 90 min.

    Card H-07 · Craniotomy¶
    Wound Class I (Clean)
    Position Supine / lateral / prone; Mayfield pins
    Avg Time 2–6 hrs
    Blade #10 scalp; #15 dural
    Implants Titanium fixation plates + screws

    Before You Scrub - Mayfield clamp + pins on field - Raney clips × 30–40 pre-loaded in applier - High-speed drill: round burr + craniotome footplate; irrigator running - Cottonoids (1/2", 3/4", 1"): ALL counted; radiopaque thread visible - Microscope sterile drape ready; CUSA if tumor case - Bipolar × 2 pairs; bone wax on field

    Closure

    Layer Suture
    Dura 4-0 Nurolon watertight
    Bone flap Titanium plates + screws
    Galea 2-0 Vicryl interrupted
    Skin Staples or 3-0 nylon

    Watch For: Cottonoid count — every one in must come out before the dura closes. Raney clips are counted items.

    Card H-08 · Lumbar Discectomy / Laminectomy¶
    Wound Class I (Clean)
    Position Prone on Wilson frame or chest rolls
    Avg Time 45–90 min
    Blade #10 midline; #15 annulotomy
    Fluoroscopy Required for level confirmation

    Before You Scrub - Wilson frame set up before patient arrives - Kerrison punches: 2, 3, 4 mm - Pituitary rongeurs: straight + angled (up-bite + down-bite) - Cobb elevator + periosteal elevators - Love nerve root retractor - Bipolar for epidural vessels

    Closure

    Layer Suture
    Thoracolumbar fascia 1-0 Vicryl figure-of-eight
    Subcutaneous 2-0 Vicryl
    Skin Staples or 3-0 Monocryl

    Watch For: Level confirmation with fluoroscopy before any bone removal — mandatory. Dural tear → clear fluid in the field = stop.

    Card H-09 · ACDF (Anterior Cervical Discectomy & Fusion)¶
    Wound Class I (Clean)
    Position Supine, shoulder roll, neck extended
    Avg Time 60–120 min
    Blade #15 skin; #11 annulotomy
    Fluoroscopy Required
    Implants PEEK cage + anterior cervical plate + screws

    Before You Scrub - Caspar retractor system: all blades + distraction pins + driver - PEEK cage or graft: one size up/down on field - Anterior cervical plate: screw lengths varied - High-speed burr (AM8); pituitary rongeurs (straight + up-bite) - Bipolar; C-arm sterile drape

    Closure

    Layer Suture
    Platysma 3-0 Vicryl interrupted
    Subcutaneous 3-0 Vicryl
    Skin 4-0 Monocryl subcuticular

    Watch For: RLN at risk with medial retraction. Confirm ETT is NIM-equipped if neuromonitoring is used. Fluoroscopy before cage placement.

    Card H-10 · CABG¶
    Wound Class I (Clean)
    Position Supine, arms tucked
    Avg Time 3–5 hrs
    Blade #10 sternotomy; #15 SVG harvest
    CPB Yes

    Before You Scrub - Sternal saw tested; Finochietto retractor on field - Sutures: 7-0 Prolene × 4 (coronary); 5-0 Prolene × 2 (proximal); 4-0 Prolene × 2 (purse-strings); stainless steel wires #5 or #6 × 8 - Chest tubes × 3–4; 2-0 silk × 4 for securing - Bone wax on field - Double-glove

    Closure

    Layer Suture
    Sternum #5 or #6 steel wires × 6–8
    Presternal fascia 0 Vicryl running
    Subcutaneous 2-0 Vicryl
    Skin Staples or 4-0 Monocryl

    Watch For: ACT >480 before cannulation. Track Prolene suture tails — keep moist. Chest tube count = part of the final count.

    Card H-11 · Carotid Endarterectomy (CEA)¶
    Wound Class I (Clean)
    Position Supine, head turned away from operative side
    Avg Time 90–120 min
    Blade #15 skin; #11 arteriotomy
    Implants Patch graft (vein, Dacron, or bovine pericardium)

    Before You Scrub - Patch graft confirmed; 6-0 Prolene × 2 on field - Silastic vessel loops × 3 (ICA, ECA, CCA) - DeBakey atraumatic clamps (straight + angled) - Freer elevator for plaque dissection - Potts scissors; heparin drawn + ready - Intraluminal shunt (Javid or Pruitt-Inahara) on field

    Closure

    Layer Suture
    Carotid sheath 3-0 Vicryl loose approximation
    Platysma 3-0 Vicryl interrupted
    Subcutaneous 3-0 Vicryl
    Skin 4-0 Monocryl subcuticular

    Watch For: Clamp time — pass instruments without delay. 6-0 Prolene breaks — have backup loaded. Flush before releasing ICA clamp.

    Card H-12 · Cesarean Section (C-Section)¶
    Wound Class II (Clean-Contaminated)
    Position Supine with left lateral tilt (wedge under right hip)
    Avg Time 30–60 min
    Blade #10 skin + fascia + uterine stab; bandage scissors for uterine extension

    Before You Scrub - Doyen (bladder blade) retractor on Mayo stand - Bandage scissors on Mayo stand (for uterine extension) - Cord clamps × 2 immediately accessible - Warm receiving towel on the lower sterile field for infant - 0 or #1 Vicryl × 2 loaded on needle holders before incision - Oxytocin: circulator has it ready for anesthesia at cord clamping

    Closure

    Layer Suture
    Uterus #1 Vicryl running (1–2 layers)
    Rectus fascia 0 Vicryl running
    Subcutaneous (if ≥2 cm) 3-0 Vicryl
    Skin Staples or 4-0 Monocryl subcuticular

    Watch For: Doyen retractor slipping = bladder risk. Uterine atony after delivery = call for uterotonics. Placenta to pathology cup.

    Card H-13 · Total Abdominal Hysterectomy (TAH)¶
    Wound Class II (Clean-Contaminated)
    Position Supine; Foley catheter in
    Avg Time 60–120 min
    Blade #10 Pfannenstiel or midline

    Before You Scrub - O'Sullivan-O'Connor retractor on field; appropriate blade sizes - Heaney clamps × 8 on Mayo stand + visible in clamp rack - 0 Vicryl ties pre-cut × 10; suture scissors immediately accessible - Long instruments: long Metzenbaum, long Deaver, long DeBakeys - Specimen cup + pathology slip ready

    Closure

    Layer Suture
    Vaginal cuff 0 Vicryl running locking
    Fascia 0 Vicryl or PDS running
    Subcutaneous 3-0 Vicryl
    Skin Staples or 4-0 Monocryl

    Watch For: Ureter at risk when clamping uterine vessels — surgeon must identify it first. Clamp slippage = call out immediately.

    Card H-14 · Cystoscopy with Ureteral Stent¶
    Wound Class II (Clean-Contaminated)
    Position Lithotomy
    Avg Time 15–30 min
    Blade None
    Fluoroscopy Required

    Before You Scrub - Cystoscope sheath + bridge + telescope (30° + 70°) - Double-J stent: confirm size ordered; unopened on field - Guidewires: hydrophilic × 1; stiff × 1 backup - Stent pusher matching the stent system - Irrigation: 1L sterile saline × 2 primed - C-arm sterile drape

    Closure No wound closure. Foley catheter may be left per surgeon preference.

    Watch For: Irrigation bottle running dry → field collapses instantly. Guidewire resistance → do not force; different wire type needed.

    Card H-15 · TURP¶
    Wound Class II (Clean-Contaminated)
    Position Lithotomy
    Avg Time 45–90 min
    Blade None
    Irrigation Glycine 1.5% (monopolar) OR saline (bipolar) — confirm before spiking

    Before You Scrub - Resectoscope: sheath (22–26 Fr) + working element + telescope — matching system - Irrigation fluid type confirmed with surgeon; do NOT mix up glycine and saline - Ellik evacuator on field - 3-way Foley (22–24 Fr, 30 mL balloon) ready for end of case - Track irrigation in vs. irrigation out throughout

    Closure No wound closure. 3-way Foley placed with continuous bladder irrigation post-op.

    Watch For: TURP syndrome (glycine absorption) — deeply negative fluid balance = emergency. Do not resect past the verumontanum.

    Card H-16 · Laparoscopic Nephrectomy¶
    Wound Class I (Clean)
    Position Lateral decubitus, operative side up
    Avg Time 90–150 min
    Blade #11 ports; #10 extraction site
    Implants None

    Before You Scrub - Endo-GIA: vascular loads (white/gray) × 3 for renal vessels - Endobag: large enough for the kidney - Kidney rest elevated and locked before prepping - 0 Vicryl for extraction site fascia closure

    Closure

    Layer Suture
    Extraction site fascia 0 Vicryl interrupted
    Port sites ≥10 mm 0 Vicryl
    Skin 4-0 Monocryl

    Watch For: Renal artery BEFORE the vein — always. Right side: IVC immediately medial to renal vein. Wrong load = critical error.

    Card H-17 · Tonsillectomy & Adenoidectomy (T&A)¶
    Wound Class II (Clean-Contaminated)
    Position Supine, shoulder roll, neck extended
    Avg Time 20–45 min
    Blade None (ESU or coblation)
    Airway Shared (RAE tube)

    Before You Scrub - Crowe-Davis mouth gag: correct size (peds vs. adult); suspension arm available - Tonsil clamps (Allis or Hurd) × 2 - ESU pencil or coblation wand — confirm generator settings - Tonsil sponges counted; specimen cups × 2 (left tonsil, right tonsil) - 3-0 chromic on a tonsil needle if suture hemostasis needed

    Closure No formal closure. Mucosal beds heal by secondary intention.

    Watch For — CRITICAL: Every tonsil sponge placed in the pharynx must come out before gag removal. A retained tonsil sponge = airway obstruction. Count out loud. Confirm before removing the gag.

    Card H-18 · Thyroidectomy (Total)¶
    Wound Class I (Clean)
    Position Supine, shoulder roll, neck extended
    Avg Time 60–120 min
    Blade #15 (Kocher incision)

    Before You Scrub - NIM ETT placed and tested before draping - Nerve stimulator (Prass probe) on field; confirmed functional - Bipolar forceps × 2 pairs: no monopolar near the RLN - Harmonic or LigaSure (confirm which device) - Small vein retractors + right-angle retractors for posterior lobe - Specimen cups × 2

    Closure

    Layer Suture
    Strap muscles 3-0 Vicryl running
    Platysma 3-0 Vicryl interrupted
    Subcutaneous 3-0 Vicryl
    Skin 4-0 Monocryl subcuticular

    Watch For: NIM silence after a dissection step = RLN may be injured — stop and confirm. Bipolar ONLY near the nerve.

    Card H-19 · Cataract (Phacoemulsification + IOL)¶
    Wound Class I (Clean)
    Position Supine, head in donut
    Avg Time 15–30 min
    Blade Keratome (2.2–2.8 mm) + paracentesis knife
    Implants IOL (confirm power + type)

    Before You Scrub - IOL loaded in injector per manufacturer instructions — do NOT touch with gloves - OVD (viscoelastic) drawn and on Mayo stand - Phaco machine: tubing primed; BSS bags spiked; footpedal set - Eye speculum: correct size - Dilating drops confirmed given 30–60 min pre-op; confirm full dilation - Every solution labeled; do NOT use CHG near the eye

    Closure Self-sealing keratome incision — no sutures. Subconjunctival antibiotic + steroid injection. Rigid eye shield (not a pressure patch).

    Watch For: IOL contamination → trash it; open new one. Posterior capsule rupture → case changes; different instruments needed. Label every solution on the back table.

    Card H-20 · Split-Thickness Skin Graft (STSG)¶
    Wound Class I–II
    Position Per wound location
    Avg Time 30–90 min
    Blade Dermatome (powered or manual)
    Two sites Donor site + recipient site

    Before You Scrub - Dermatome: blade guard depth set per surgeon specification; mineral oil on field - Mesh carrier (if meshing): correct expansion ratio assembled - Damp saline gauze on back table for graft storage - Bolster or NPWT dressing confirmed - Donor site dressing: Adaptic/Xeroform + gauze + ACE - Change outer gloves between donor harvest and recipient site

    Closure Donor site: non-adherent dressing only — NOT sutured. Graft secured with staples + bolster or NPWT.

    Watch For: Graft desiccation → keep it moist at all times. Hematoma under graft = graft failure. Achieve complete hemostasis in the recipient bed before graft goes down.

    Card H-21 · Mastectomy with Implant Reconstruction¶
    Wound Class I (Clean)
    Position Supine, operative arm on arm board
    Avg Time 2–4 hrs
    Blade #10 (elliptical periareolar incision)
    Implants Tissue expander or DTI silicone implant + ADM

    Before You Scrub - Implant or expander: confirm type and size; keep sealed until pocket irrigated - Triple antibiotic solution (gentamicin + bacitracin + polymyxin B) for pocket irrigation - Change outer gloves before handling implant — no exceptions - ADM: keep in saline until needed; do not allow it to dry - JP drains × 2; 2-0 nylon for securing - Lighted retractor or fiberoptic headlight for flap elevation

    Closure

    Layer Suture
    Pectoralis / ADM 2-0 or 3-0 Vicryl
    Subcutaneous 3-0 Vicryl
    Skin 4-0 Monocryl subcuticular

    Watch For: No-touch technique — implant cannot contact skin. No compression on the implant after placement.

    Card H-22 · Pyloromyotomy (Pediatric)¶
    Wound Class I (Clean)
    Position Supine, small shoulder roll
    Avg Time 20–40 min
    Blade #15 × 2
    Patient ~2–8 weeks old; handle everything with extra care

    Before You Scrub - Baby-scale instruments: Denis Browne retractors, Fredet-Ramstedt spreader or fine hemostat - Warming blanket active; warm irrigation on field; room temp elevated - 3-0 Vicryl fascia; 4-0 Vicryl subcutaneous; 5-0 Monocryl skin - Mucosal integrity test: anesthesiologist injects air through NGT at end; saline in wound; watch for bubbles

    Closure

    Layer Suture
    Fascia 3-0 Vicryl interrupted
    Subcutaneous 4-0 Vicryl
    Skin 5-0 Monocryl or Dermabond

    Watch For: Mucosal perforation — repair + Graham patch if it occurs. Do not cross the pyloric vein distally. Temperature drop in this patient = emergency.

    Card H-23 · CDH Repair (Congenital Diaphragmatic Hernia)¶
    Wound Class I (Clean)
    Position Supine, slight contralateral rotation
    Avg Time 60–120 min
    Blade #15 (subcostal incision)
    Patient Neonate (days old); left side most common

    Before You Scrub - Neonatal instrument set (baby-scaled only) - 2-0 or 3-0 Prolene/Ethibond interrupted for primary repair - Gore-Tex patch on field if large defect expected - Warming blanket active; warm irrigation; overhead warming lamp if available - Check blood loss per sponge — 10 mL = >4% of a 3 kg neonate's blood volume

    Closure

    Layer Suture
    Fascia 2-0 or 3-0 Vicryl interrupted
    Subcutaneous 3-0 Vicryl
    Skin 5-0 Monocryl or skin glue

    Watch For: Temperature every 15 min — a 2°C drop is a physiological emergency. Herniated organs are fragile; handle with extreme gentleness during reduction.

    Card H-24 · Free Flap (Microvascular)¶
    Wound Class I (Clean)
    Position Varies by flap and defect
    Avg Time 4–10 hrs
    Blade #15 both sites
    Two teams Donor harvest + recipient site simultaneously

    Before You Scrub - Microvascular instruments: Castroviejo needle drivers × 2; jewelers forceps × 2; micro scissors; vessel background material - 9-0 or 10-0 nylon on BV-1 needle × multiple (keep moist) - Venous coupler system (confirm sizes) - Sterile Doppler probe - Operating microscope drape - Communication: both surgeons must confirm readiness before pedicle division

    Closure Donor site: layered primary closure or STSG. Recipient: non-compressive window dressing only — no pressure on the flap.

    Watch For: Ischemia time begins at pedicle division — have everything ready before that moment. Compression on the flap after anastomosis = thrombosis = flap loss.

    Card H-25 · da Vinci Robotic Surgery (Bedside Assistant)¶
    Platform da Vinci Xi / X / SP (Intuitive Surgical)
    Your Station Patient cart (bedside)
    Surgeon Location Console — non-sterile
    Your Port 10–12 mm assistant port

    Before You Scrub - Robotic arm drapes × 4: applied before docking; no wrinkles; confirm locked - EndoWrist instruments: check use counter — expired instruments cannot be loaded - Assistant port instruments: laparoscopic clip applier, suction-irrigator, grasper, endobag - Pre-load anastomosis sutures on laparoscopic needle holders 20 min before the suturing step - Instrument exchange protocol memorized: release button → pull straight out → seat new instrument → confirm on monitor

    Watch For: - Pre-load every suture before it is called — the console surgeon cannot wait - Never grab a robotic arm while it is under power — significant torque risk - Arm drape breach → stop; re-drape the affected arm before continuing - Instrument counter expiration → open replacement immediately

    Card H-26 · Mako SmartRobotics (Stryker)¶
    Platform Mako SmartRobotics (Stryker)
    Procedures THA, TKA, UKA
    Requires Pre-op CT plan loaded before case
    Cutting Tool Burr (not a saw)
    Implant System Stryker (Triathlon knee / Accolade hip)

    Before You Scrub - Pre-op CT plan confirmed loaded on Mako system - Tibial + femoral tracker arrays (sterile; pin-mounted) on field - Registration probe: sterile; immediately available when surgeon calls - Mako burr loaded and sterile; backup burr on field - Irrigation running during burr resection (heat generation) - All trial components in order; cement supplies ready (Mako guides cuts; cementing is still manual)

    Watch For: - Array disturbance = registration lost → re-register required - Burr requires continuous irrigation — confirm before activation - Arrays are counted; note their position to the whole team before draping

    Card H-27 · ROSA (Zimmer Biomet)¶
    Platform ROSA Knee / ROSA Hip (Zimmer Biomet)
    ROSA Knee Imageless — no pre-op CT required
    ROSA Hip Image-based — pre-op CT required
    Cutting Tool Saw (ROSA positions the cutting jig)
    Implant System Zimmer Biomet (Persona, NexGen)

    Before You Scrub - ROSA system powered on; sterile arm drape applied - Bone pins + tracker arrays: sterile; on field - Registration probe: sterile - ROSA saw jig adapter confirmed compatible with implant system - Standard TKA instruments still required for manual steps

    Watch For: - Surgeon will spend time at the ROSA screen building the intraoperative plan — do not pass instruments during this phase - Saw blade must be fully seated in the jig before activation - ROSA arm drape breach = arm exits the sterile field

    Card H-28 · Mazor X (Medtronic)¶
    Platform Mazor X (Medtronic)
    Procedure Spine pedicle screw placement
    Requires Pre-op CT plan + O-arm spin for registration
    Robot Does Positions drill guide at planned trajectory
    Surgeon Does Drills, taps, and places screws manually
    Implant System Medtronic (CD Horizon, Solera)

    Before You Scrub - Pre-op CT plan confirmed loaded - Spine reference array (sterile; clamp-mounted on spinous process or iliac crest) - Mazor arm sterile drape applied before arm enters field - Mazor-specific drill guide cannulas on field - O-arm sterile drape ready - Medtronic screw set: confirm sizes; set screws in their rack (counted items)

    Watch For: - Reference array movement = invalidated registration → alert surgeon immediately - O-arm spin = hands off patient, everyone step back, call it loudly - Drill guide must contact cortical bone before drilling — not soft tissue - Set screws are tiny counted items; never leave unsupported on the field

    Card H-29 · ExcelsiusGPS (Globus Medical)¶
    Platform ExcelsiusGPS (Globus Medical)
    Procedure Spine pedicle screw placement
    Unique Feature Integrated navigation + robotics (one platform)
    Can work with or without pre-op CT Intraoperative O-arm registration available
    Implant System Globus Medical (CREO, REVOLVE) — NOT interchangeable with Medtronic

    Before You Scrub - Workflow confirmed: pre-op CT plan OR intraoperative O-arm registration - ExcelsiusGPS arm sterile drape applied - Globus-specific reference array and drill cannulas on field - Globus screw set confirmed (not Medtronic, not DePuy) - O-arm or C-arm sterile drape ready

    Watch For: - ExcelsiusGPS arm locks automatically at planned trajectory — confirm lock indicator displayed before drilling - Reference array = same rules as Mazor X; any movement requires re-registration - O-arm spin protocol: full room clear, hands off patient

    Card H-30 · Stealth Navigation (Medtronic)¶
    Platform Stealth Navigation (Medtronic)
    Procedure Spine, cranial, ENT navigation guidance
    Robot None — navigation only (no robotic arm)
    Tracking Optical (camera must have line of sight to trackers)

    Before You Scrub - Stealth reference array: sterile; clamp-mounted; placed before O-arm spin - Instrument trackers (sterile): attach to drill, pedicle probe, tap — confirm all are on the field - Stealth-compatible instruments only: non-tracked instruments won't appear on the screen - O-arm sterile drape if spin is planned - Tracker ball cleaning routine: wipe balls with a dry sponge every time the instrument returns to the back table

    Watch For: - Tracker ball contamination = instrument disappears from navigation screen silently - Line-of-sight blocking: do not cross between the Stealth camera and the tracker arrays - O-arm spin = full room clear; hands off field; call it every time - If navigation fails intraoperatively, case continues with fluoroscopy — have C-arm sterile drape in the room
