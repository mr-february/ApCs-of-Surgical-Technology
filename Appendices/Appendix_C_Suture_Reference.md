## Needle Type Quick Reference

| Point Type | Symbol/Code | Cross-Section | Use |
|-----------|------------|--------------|-----|
| Taper (round) | RB, SH, MH, CT | Round | Soft tissue: bowel, fascia, vessel |
| Conventional cutting | CE | Triangular (cutting edge inside curve) | Skin, tough tissue |
| Reverse cutting | CE-2, PS, FS | Triangular (cutting edge outside curve) | Skin, fibrous tissue (preferred) |
| Tapercut | TC | Taper body + cutting tip | Calcified tissue, fascia |
| Blunt | BL | Blunt rounded | Liver, kidney, blunt dissection |
| Spatula | | Flat with cutting edges | Ophthalmic, superficial layers |
